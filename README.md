## البدء السريع

```bash
# 1. نسخ ملف البيئة
cp .env.example .env

# 2. إضافة مفاتيح Supabase في .env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

# 3. تثبيت المكتبات
npm install

# 4. تشغيل التطوير
npm run dev

# 5. البناء للنشر
npm run build
```
